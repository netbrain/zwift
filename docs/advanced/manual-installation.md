# Manual Installation Steps

## How can I build the image myself?

```bash
./src/build-image.sh
```

## How can I fetch the image from docker hub?

<https://hub.docker.com/r/netbrain/zwift>

```bash
docker pull netbrain/zwift:$VERSION # or simply latest
```
